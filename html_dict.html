Dicionário de Tags HTML

!DOCTYPE html
Define o tipo de documento e a versão do HTML usada.
html
Elemento raiz de um documento HTML.
head
Contém metadados da página como título, codificação e links de estilo.
meta
Define informações de metadados, como charset (conjunto de caracteres).
title
Define o título da aba do navegador.
link
Utilizado para vincular arquivos externos, como CSS ou ícones.
body
Contém todo o conteúdo visível da página.
div
Contêiner genérico para agrupar conteúdos e aplicar estilos.
aside
Define uma seção lateral, como uma barra lateral de navegação ou informações extras.
header
Define o cabeçalho da página ou de uma seção.
nav
Representa um menu de navegação.
ul
Lista não ordenada.
li
Item de lista.
main
Define o conteúdo principal da página.
section
Define uma seção temática da página.
h1 até h6
Define títulos de diferentes níveis, sendo h1 o mais importante e h6 o menos.
p
Parágrafo de texto.
a
Cria um link para outra página ou local da mesma página.
img
Insere uma imagem na página.
strong
Indica que o texto tem importância forte, geralmente mostrado em negrito.
br
Insere uma quebra de linha.
footer
Define o rodapé da página.
dl
Lista de definição, usada para pares termo-descrição.
dt
Termo a ser definido em uma lista de definição.
dd
Descrição de um termo em uma lista de definição.
